## Getting started

Set the environment
```bash
python3.12 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

By using `uv`
```bash
uv venv .venv
source .venv/bin/activate
uv pip install -r requirements.txt
```
